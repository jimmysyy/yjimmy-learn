# 10-08:

​ 国庆结束了，校图书馆也开门了，走了三个楼层，只看到了一个学生在学习。随便坐下的座位上也有了一层薄灰，疫情把这栋楼的生气都给剥夺了。
​ 去信用社激活社保卡，因为学校的费用都打进社保卡里了，询问后得知被冻结了，因为诈骗频发，而这张卡我从来没使用过，是强制办理的，没想到就这样冻结了。我想笑又笑不出来，因为我们学校被骗几十万金额的就有好几位同学，甚至前几天关注的一个博主在多次认定对面为假警察的情况下，还是被骗了几十万，防不胜防。我最多只能给父母去个电话，提醒他们多注意。

# 10-12

​ 几天了还是默认状态，疫情下的生活比中学周末假期还让人无聊。每天就是起床核酸，吃饭，睡觉。前一项比后两项重要，目前来说。

​ 昨天也聊了一下当前情况，学校因为疫情不给离校，如果离校就不允许回来。而投的简历需要线下面试，当前求职环境又不理想，如果我破釜沉舟离开学校去省会面试，可能需要隔离之后面试，面试不通过后因为职位稀少的原因又不能在当地待太久，结果就是两个选择，回家居家隔离十四天，申请回校报备后也需要隔离。一场几百公里外的面试来回需要的时间可能就是一个月。这对于古人来说都是难以想象的吧。

# 10-14

​ 今天修改简历后在boss上尝试投了几个省内公司，都是已读不回，欸，都不知道该抱怨什么了。

​ 晚上学校又骗人开会，五分钟不到就图穷匕见了，是两个培训公司来宣传。一个是新华三培训，不花钱培训花钱考试，号称证书华为思科等皆认同，我也不知道华为和思科自己的证书跟这个是什么级别吧。另一个是网安培训，班里三分之一的同学好像都去学这个了，但是这个公司好像招不够，来宣传几次了，费用非常非常高，不过培训应该培训的很好，因为即使是同学也问不出一点口风，毕竟是学网安的嘛，安全意识高正常。

​ 我提这个会议的是因为它们的宣传，画的饼挺大，待遇说很好，工资说很高。跟上面投递招聘各种已读不回对比挺奇怪的，是挺奇怪的，听说今年毕业生的灵活就业占比很大。可能上面两个公司在下一个风口吧，就这样。

# 10-17

​ 又是几天没写，今天单词也没背，有点摆烂了。开学政策不明确，开大会的背景下又严肃了。想到开学后的网格化管理，欸。

​ 不过今天把牛客网的前端面试题库整理了，也算做点事了。牛客网面试题库

# 10-19

​ 跟家里交流后，还是决定先回家了，还有十天就要开学了，对于没有课的我来说，网格化管理 = 坐牢。所以我必须在十天内离开这里，但是我的显示器和台式电脑再一次给我创造困难，国内的快递我已经找不到敢快递电脑和显示屏的了，即使你有原包装，即使你保价，你都不能让站点邮寄，因为他们也知道他们邮递过程中的暴力运输，他们知道一定会损坏你的设备。

# 10-20

​ 接着昨天回家这件事，今天导员通知可以回原宿舍拿衣服，我也打算回去拿行李，结果到了门口被宿管阻拦，即使多次请求还是不放行一步。我也不想多说，又是疫情下的层层加码。

​ 这种男宿管就是跟某些物业保安一般，当他们有了一捏捏的权力，他们就敢面向他们的更弱者张牙舞爪，他们会竭尽所能展示他们那渺小的权力，而且在当前的大背景下，他们的理由是如此的理直气壮，“你不能干扰防疫懂不懂啊，出了事你负责的起吗”。

​ 我觉得这个每日一记又可以写一些新东西了，我想知道当前背景啊，小鬼究竟有多难缠。

​ 当然我的回家计划又耽搁几天了，把我困死在这吧。

# 10-22

​ 今天回到宿舍拿来了行李，看来可以预定回家的日期，躺在床上想到三个月都呆在一栋楼内，对我实在是太不可思议了。

​ 最近的一个月，可以明确我的心态已经有变化了，我为了获取糖带来的愉悦，已经对饮料上瘾了，即使在每天正常喝白开水的情况下，仍然需要一瓶高糖饮料来满足自己。

# 10-26

​ 回家已经快两天了，一切都是老样子，就像十年前，也可能是我没出门的原因。

​ 后果第一次出门就被锁在门外，家里把锁换了而又没通知我，只能让爸大老远回来给我送钥匙。

​ 不过回家后的心情自然是不同往日，这种放松自然的心态太舒服了，我回家就是为了这个。

​ 哦，还有令人厌烦的三日核酸。
