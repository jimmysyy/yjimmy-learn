# 一、什么是Node.js？

Node.js 是一个基于 Chrome V8 引擎 的 JavaScript 运行时环境

Nodejs官网

  • 浏览器是Javascript的前端运行环境
  • Node.js是JavaScript的后端运行环境
  • Node.js中无法调用DOM和BOM等浏览器内置API

Node.js可以做什么

  1. 基于Express框架open in new window，可以快速构建Web应用
  2. 基于Electron框架open in new window，可以构建跨平台的桌面应用
  3. 基于restify框架open in new window，可以快速构建API接口项目
